# Can you find the CVE [500 pts]

**Category:** network
**Solves:** 1

## Description
>Divisi Keamanan Siber di Badan Pengawasan Keuangan (BPK) berhasil mendeteksi aktivitas mencurigakan dalam jaringan internal mereka. Sebuah serangan yang tidak biasa terdeteksi melalui alat monitoring, dan tim forensik berhasil mengekstrak file log jaringan dalam format .pcap.\r\n\r\nDari analisis awal, diketahui bahwa serangan tersebut memanfaatkan celah keamanan pada salah satu aplikasi atau protokol yang digunakan. Namun, hingga saat ini, tim belum berhasil mengidentifikasi CVE ID (Common Vulnerabilities and Exposures) yang digunakan oleh pelaku.\r\n\r\nSebagai seorang analis keamanan siber, Anda ditugaskan untuk memecahkan kasus ini. Tugas Anda adalah mengidentifikasi CVE ID dari kerentanan yang dieksploitasi dengan menganalisis file PCAP yang telah diberikan. Gunakan keahlian Anda dalam membaca log jaringan, mendeteksi pola serangan, dan memahami eksploitasi kerentanan.\r\n\r\nFormat flag BPKCTF{CVE-202X-XXXX}
